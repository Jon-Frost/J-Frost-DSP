# J-FROST-DSP

This project is a Flask-based data dashboard application that lets users upload datasets, preview and clean data, build interactive charts, save dashboards, export dashboard views, and use AI-assisted insights for chart suggestions and data exploration.

## HOW TO RUN

1. Open a terminal in the DSP-Project folder.
2. Create a Python virtual environment.
3. Activate the virtual environment.
4. Install the required packages from requirements.txt.
5. If needed, add a GEMINI_API_KEY value to a .env file for AI chat features.
6. Run the Flask app with `python app.py`.
7. Open your browser and go to http://127.0.0.1:5000.
8. Register or log in, then upload a dataset and start building dashboards.
